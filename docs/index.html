ADO Pages — Sample Pages

Example HTML pages rendered by the ADO Pages Chrome extension

About

ADO Pages is a Chrome extension that renders HTML files in Azure DevOps repositories as live webpages. These sample pages demonstrate the kind of content that works well with the extension.

Tools

Windows Error Code Lookup

Search HRESULT, NTSTATUS, and Win32 error codes by hex value, decimal, name, or description. Decodes HRESULT facility/code bits and converts Win32 codes to their HRESULT equivalents.

Base Converter

Convert between hex, decimal, octal, and binary with 8/16/32/64-bit width support. Includes a bit viewer, byte-order display, and quick access to common Windows constants.

Documentation Samples

API Reference

A styled REST API documentation page with method badges, parameter tables, request/response examples, and error handling reference.

Architecture Diagram

A system design document with an inline SVG architecture diagram, component descriptions, health status indicators, and deployment details.

Try it yourself

  1. Install the ADO Pages extension
  2. Push HTML files to any Azure DevOps Git repository
  3. Navigate to an .html file and click "View as Webpage ↗"
